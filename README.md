## Projects

### 🌸 [Iris Flower Classification — Random Forest](https://github.com/khanzalan/iris-random-forest)
> Multi-class classification of iris species using a Random Forest model trained on sepal and petal measurements.

- Built full ML pipeline: EDA → preprocessing → model training → evaluation
- Applied `StandardScaler` for feature normalization and `joblib` for model persistence
- Visualized feature importance and confusion matrix for model interpretability
- **Stack:** Python · scikit-learn · Pandas · Seaborn · Matplotlib

---

### 🚗 [Car Price Prediction — Linear Regression](https://github.com/khanzalan/car-price-prediction)
> Regression model predicting used car selling prices from a dataset of 300+ vehicles across brands like Maruti, Toyota, Hyundai, and Honda.

- Performed feature selection, outlier analysis, and categorical encoding
- Evaluated model using RMSE, MAE, and R² metrics
- **Stack:** Python · scikit-learn · Pandas · Matplotlib
